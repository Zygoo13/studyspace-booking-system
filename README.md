# StudySpace

StudySpace is a web application for managing and booking study/work spaces in a café.

## Tech Stack

* Backend: Spring Boot, MySQL
* Frontend: React + TypeScript

## Project Structure

studyspace/

* backend/
* frontend/
* docs/

## Run (local)

### Backend

cd backend
./mvnw spring-boot:run

### Frontend

cd frontend
npm install
npm run dev
